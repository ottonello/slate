    Get existing extensions

    GET /admin/v1/extensions HTTP/1.1

    HTTP/1.1 201 Created
Content-Type: application/json

{
    "extensions": [
        {
            "id": "string",
            "owner": "string",
            "api-key": "string",
            "profile": {
                "name": "string",
                "picture": "string",
                "url": "string"
            },
            "config": {
                "enableWebhooks": "boolean",
                "webhooks": {
                    "message": "string"
                }
            }
        }
    ]
}

    HTTP/1.1 400 Bad Request
Content-Type: application/json

{
    "code": "integer",
    "status": "string",
    "message": "string"
}

    HTTP/1.1 500 Internal Server Error
Content-Type: application/json

{
    "code": "integer",
    "status": "string",
    "message": "string"
}

    Parameters

    Name In Type Description
    limit path integer Optional. The total number of extensions to get in this page.
    start path integer Optional. The index of the first extension in the page.

    Responses

    Http code Type Description
    201 Extensions extension successfully created
    400 Error invalid request parameters
    500 Error internal error
